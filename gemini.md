# Global Rules & Principles

## Tech Stack
- **Framework:** React + Vite
- **Routing:** React Router v6
- **Styling:** Vanilla CSS (Vars, Modules), No Tailwind CSS
- **Icons:** Lucide React
- **State Management:** TBD (Context API or Zustand for simplicity)

## Brands & Entities
- **AI Agent:** Mick (Orchestrator, always-on)
- **Company:** Icarus Digital Marketing
- **Brands:**
  - **VERO:** Peptide brand
  - **Lumova Health**

## Design Principles
- **Aesthetic:** High-end, premium, modern, polished web design. Look and feel extremely premium, avoiding generic colors and styles.
- **Theme:** Dark Mode Only.
- **Layout:** Clean cards with generous spacing. Desktop Optimized only.
- **Colors:** Deep, rich dark backgrounds with subtle accent colors tailored per brand/entity:
  - **Icarus:** Blue
  - **VERO:** Green
  - **Lumova:** Purple
  - **Admin/General:** Gray
- **Typography:** Modern, highly legible fonts (e.g., Inter, Outfit) with excellent hierarchy.
- **Interaction:** Dynamic elements, hover effects, subtle glassmorphism, smooth micro-animations for an interface that feels responsive and alive.
- **UX:** Clear sidebar navigation on the left, main content area taking up the rest of the layout.
